# **ISLP Applied Solutions**

## **Project Description**

This repository contains solutions to all applied problems from the book ***An Introduction to Statistical Learning with Applications in Python (ISLP)***. The edition used is the **first printing (July 5, 2023)**.
You can access the free book and additional resources at: [https://www.statlearning.com/](https://www.statlearning.com/)

**Note 1:** Only the Jupyter notebooks are included here. The full project directory, containing all datasets and auxiliary files, is quite large and therefore have been modified.

**Note 2:** At times, I intentionally avoid using the book’s custom packages and helper functions in order to deepen the learning experience.
